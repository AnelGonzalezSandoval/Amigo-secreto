# Amigo-secreto
Juego para sortear un amigo secreto. Con éste código se crea una página en la que el usuario puede ir adicionando nombres de amigos, uno a uno, después puede sortear y le sale de forma aleatoria el nombre de un amigo, su "Amigo secreto".
La página se mostrará de la siguiente manera:
<img width="956" height="769" alt="image" src="https://github.com/user-attachments/assets/212451db-90d2-4923-a730-869015204fd6" />
En el campo Escriba un nombre, deberá escribir uno a uno los nombres, escribe un nombre y da clic en Añadir:
<img width="832" height="240" alt="image" src="https://github.com/user-attachments/assets/a56c2f7e-dac1-4a78-9b95-37d7ddbd4afc" />
Una vez que de clic en Añadir, se empezará a generar la lista de amigos en la parte de abajo del cuadro de texto:
<img width="805" height="372" alt="image" src="https://github.com/user-attachments/assets/5f9ec49a-b047-4918-bc00-995120daabfd" />
Puede agregar tantos amigos como desee, una vez terminada su lista de amigos, deberá dar clic en el botón sortear amigo, y le aparecerá en Verde y más grande el amigo secreto:
<img width="972" height="799" alt="image" src="https://github.com/user-attachments/assets/c654a7a1-83e7-4fa0-915a-bcb528780306" />
Cada que usted vuelva a iniciar el juego, le sorteará de manera aleatoria el amigo secreto.

Si el campo para añadir el nombre está vacío y da clic en Añadir, le aparecerá una alerta para que ingrese un nombre válido:
<img width="953" height="644" alt="image" src="https://github.com/user-attachments/assets/8d570167-e5f1-44aa-aaa1-6e0594a5c766" />
Si intenta sortear al amigo secreto sin haber ingresado ningún nombre de amigo y clicado Añadir, le aparecerá una alerta para que ingrese al menos un amigo:
<img width="1025" height="651" alt="image" src="https://github.com/user-attachments/assets/cecfeb71-1db8-4345-8199-29beef845422" />

Esperamos que disfrute sorteando su Amigo Secreto!
